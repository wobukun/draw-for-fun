# 抽卡模拟器v0.4.1

## 项目概述
抽卡模拟器是一个基于Web的应用程序，模拟流行游戏中的抽卡机制，模拟角色和武器的抽取。
该模拟器使用的概率模型参数不能保证完全准确，数据结果仅供娱乐，抽取结果与现实无关。

## 核心功能
- **角色抽卡**：基础5星概率0.6%，90抽必出5星，50%概率UP角色，非UP则下次必UP
- **武器抽卡**：基础5星概率0.7%，80抽必出5星，75%概率UP武器，非UP则下次必UP
- **武器定轨**：UP武器50%概率为定轨，非定轨加命定值，命定值1时必出定轨，抽中后清零
- **多种抽取**：支持单抽、十连和自动批量模拟
- **详细统计**：抽取结果的数学统计，包括平均抽数、中位数等
- **历史记录**：显示最近10次5星抽取记录

## 技术架构
- **后端**：Flask RESTful API服务器
- **前端**：Vue.js交互式Web界面
- **数据处理**：Python概率计算和统计分析
- **性能优化**：限制数据渲染量，优化页面跳转逻辑

## 快速开始

### 先决条件
- Python 3.12+
- Node.js和npm

### 安装
```bash
npm install
```

### 运行
```bash
python backend/main.py
```

自动启动：
- Flask后端服务器（端口：8888）
- Vue前端开发服务器（端口：3000）
- 打开浏览器到前端界面

## API端点
- **POST /api/wish**：处理祈愿请求
- **POST /api/shutdown**：关闭服务器
- **GET /api/**：返回服务器状态
